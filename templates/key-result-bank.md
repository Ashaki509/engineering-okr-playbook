# Key Result Bank - Engineering OKRs

Library of proven, high-quality Key Results organized by domain.

---

### Platform & Infrastructure

- Increase p99 API response time from Xms to <Yms
- Reduce infrastructure cost per user by Z%
- Achieve 99.99% uptime across Tier 1 services
- Reduce mean time to recovery (MTTR) from X to Y hours
- Complete migration of X services to new platform

### Developer Productivity

- Reduce average cycle time from code commit to production from X to Y days
- Increase deployment frequency from X to Y per week
- Reduce onboarding time for new engineers from X weeks to Y days
- Achieve >90% code coverage on critical services

### Security & Compliance

- Reduce open critical vulnerabilities from X to <Y
- Achieve 100% completion of quarterly security training
- Implement zero-trust architecture for X key systems

### Quality & Reliability

- Reduce production incident count by X%
- Decrease error rate on key user journeys from X% to Y%
- Improve customer satisfaction score (CSAT) from X to Y

---

**Tips:** Adapt to your context. Make them time-bound and measurable.
